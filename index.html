⚡

DLSS 4.5 — 멀티 프레임 생성

6배 멀티 프레임 생성으로 극도로 부드러운 화면을 구현하며, 동적 멀티 프레임 생성으로 목표 프레임 속도에 실시간으로 맞춥니다.

🧠

신경망 실시간 렌더링

실사 같은 조명과 재질을 실시간으로 적용하는 신경망 렌더링 모델로 현실과 가상의 경계를 허뭅니다.

🎬

DLSS 5 — 할리우드급 그래픽

렌더링과 현실 사이의 간극을 메우며, 이전에는 특수효과에서만 가능했던 수준의 사실적인 컴퓨터 그래픽을 게임에 구현합니다.

🎮

동적 프레임 배율

게임 및 NVIDIA 앱이 게임 플레이 중 프레임 배율을 조정하여 디스플레이 새로 고침 빈도에 완벽하게 맞춥니다.

기능 상세

기능 더보기

6배 멀티 프레임 생성(MFG)은 AI가 실제 렌더링 프레임 사이에 추가 프레임을 생성하여 출력 프레임 레이트를 극적으로 높입니다.
  • 단순 2배가 아닌 6배 배율로 극도로 부드러운 모션 구현
  • 동적 멀티 프레임 생성: 실시간으로 배율을 조절해 V-Sync 및 목표 FPS 유지
  • 게임·NVIDIA 앱 레벨에서 플레이어가 원하는 프레임 레이트에 자동 적응
  • 레이턴시 최소화를 위한 NVIDIA Reflex와 통합
RTX 50 시리즈는 신경망 기반 실시간 렌더링을 통해 기존 래스터라이제이션의 한계를 돌파합니다.
  • 셀 단위로 실사 조명, 전역 조명, 재질 반응을 신경망으로 추론
  • 기존 레이 트레이싱 대비 훨씬 낮은 연산 비용으로 유사한 품질 달성
  • 게임 씬의 복잡도와 무관하게 일관된 품질 유지
  • 크리에이터 워크플로에도 동일하게 적용 가능
DLSS 5는 렌더링 품질의 새로운 패러다임을 제시합니다.
  • 이전까지 할리우드 특수효과 파이프라인에서만 구현 가능하던 사실적 CG를 실시간 게임에 구현
  • 렌더링과 현실의 시각적 간극을 최소화하는 뉴럴 셰이딩
  • 게임 개발자가 더 적은 폴리곤으로 더 사실적인 결과물 생성 가능
  • 차세대 AAA 타이틀에서 실질적인 영화급 퀄리티 게임플레이 경험
